Svømmeklokke2 Documentation
============================

**Svømmeklokke2** is a web-based analog pace clock for swim training. Display timing for swimming intervals on large screens with highly customizable appearance.

Quick Start
-----------

* **Hosted Clock**: https://arildj78.github.io/PaceClock/clock.html
* **Pool Display**: Open ``index.html`` for dual-panel view (clock + workout slides)
* **Standalone Clock**: Open ``clock.html`` for full-screen clock
* **Live Configuration**: Add ``?setupmode=1`` to adjust settings in real-time
