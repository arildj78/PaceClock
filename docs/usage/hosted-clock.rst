Hosted Clock
============

The pace clock is hosted on GitHub Pages and available immediately without installation.

**Access the clock here:**

https://arildj78.github.io/PaceClock/clock.html

You can use this link from any device with a web browser. For best results, use it on large displays or projectors during pool training sessions.

**Features of the hosted version:**

* No installation required
* Always up-to-date with latest improvements
* Full customization via URL parameters
* Works offline after initial page load (uses system time only)

**Bookmark or share the URL**

You can bookmark this URL in your browser or share it with your team. All configuration is stored in the URL itself, making it easy to create and share custom setups.

**Local alternative**

If you prefer to host the clock locally, you can clone the repository or download the files and open ``clock.html`` directly in a web browser.
