Numbers
=======

Configure the clock face numbers (5, 10, 15, 20, 25, 30, 35, 40, 45, 50, 55, 60).

===================  =========  =============================================
Parameter            Default    Description
===================  =========  =============================================
``numberRadiusRel``  0.63       Radial distance of numbers from center
``numberFontRel``    0.229      Font size for clock numbers
===================  =========  =============================================

Individual Number Fine-Tuning
------------------------------

Each number can be individually adjusted with offset parameters:

* ``num5dx``, ``num5dy`` - Horizontal and vertical offset for "5"
* ``num10dx``, ``num10dy`` - Offsets for "10"
* ``num15dx``, ``num15dy`` - Offsets for "15"
* ``num20dx``, ``num20dy`` - Offsets for "20"
* ``num25dx``, ``num25dy`` - Offsets for "25"
* ``num30dx``, ``num30dy`` - Offsets for "30"
* ``num35dx``, ``num35dy`` - Offsets for "35"
* ``num40dx``, ``num40dy`` - Offsets for "40"
* ``num45dx``, ``num45dy`` - Offsets for "45"
* ``num50dx``, ``num50dy`` - Offsets for "50"
* ``num55dx``, ``num55dy`` - Offsets for "55"
* ``num60dx``, ``num60dy`` - Offsets for "60"
