Hands & Diamond
================

Configure the appearance of the clock hands and their diamond-shaped pointers.

Second Hand (Main Hand)
-----------------------

========================  =========  =============================================
Parameter                 Default    Description
========================  =========  =============================================
``shaftWidthRel``         0.04       Width of the hand shaft
``tipRel``                0.986      Length of hand (tip distance from center)
``diamondBaseInsetRel``   0.247      Position of diamond base along hand
``diamondBackRel``        0.025      Inner extent of diamond (toward center)
``diamondWidthRel``       0.098      Width of diamond pointer
``handColor``             #ffffff    Color of second hand (hex color code)
========================  =========  =============================================
