Minute Hand
============

Optional minute hand for tracking longer intervals (completes one rotation per hour).

* ``showMinute`` (1) - Display minute hand (0=off, 1=on)
* ``minuteShaftWidthRel`` (0.015) - Width of minute hand shaft
* ``minuteTipRel`` (1.000) - Length of minute hand
* ``minuteDiamondBaseInsetRel`` (0.058) - Position of diamond base
* ``minuteDiamondBackRel`` (0.135) - Inner extent of diamond
* ``minuteDiamondWidthRel`` (0.112) - Width of diamond pointer
* ``minuteColor`` (#444444) - Color of minute hand
