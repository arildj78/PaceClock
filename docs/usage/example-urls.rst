Example URLs
=============

Here are practical examples showing how to create URLs for different use cases.

Minimal Display (No Digital Time)
---------------------------------

``https://arildj78.github.io/PaceClock/clock.html?vfdEnabled=0``

Clean analog-only clock without digital time display.

Large Clock with Logo
---------------------

``https://arildj78.github.io/PaceClock/clock.html?outerRel=0.98&showLogo=1&logoSizeRel=0.15``

Maximizes clock size and displays club logo at larger scale.

Green Display Theme
-------------------

``https://arildj78.github.io/PaceClock/clock.html?vfdColor=%2300ff00&handColor=%2300ff00``

Changes digital display and hand color to green (hex color #00ff00).

Seconds Only (No Minute Hand)
-----------------------------

``https://arildj78.github.io/PaceClock/clock.html?showMinute=0``

Disables the minute hand for short interval training.

Fast Drift for Screensaver
--------------------------

``https://arildj78.github.io/PaceClock/clock.html?driftEnabled=1&driftSpeedRel=0.001``

Enables visible drift movement across the screen (50x default speed).

Custom Size and Position
------------------------

``https://arildj78.github.io/PaceClock/clock.html?outerRel=0.85&logoSizeRel=0.14&logoOffsetYRel=0.35``

Medium clock size with logo positioned lower on the face.

Presentation Mode (Static Hand Position)
----------------------------------------

``https://arildj78.github.io/PaceClock/clock.html?runClock=0&manualAngle=45``

Stops clock animation and sets hand to 7.5 seconds (45 degrees) for screenshots or demonstrations.

Setup Mode for Configuration
----------------------------

``https://arildj78.github.io/PaceClock/clock.html?setupmode=1``

Opens the configuration panel to adjust all settings interactively.

Complete Custom Configuration
-----------------------------

``https://arildj78.github.io/PaceClock/clock.html?outerRel=0.92&showLogo=1&logoSizeRel=0.13&vfdEnabled=1&vfdColor=%2300b7ff&showMinute=1&minuteColor=%23666666&driftEnabled=1``

Full customization: specific size, logo, cyan digital display, grey minute hand, and drift enabled.
