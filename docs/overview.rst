Project Overview
================

Components
----------

1. **index.html** - Dual-panel display with clock (left) and Google Slides (right)
2. **clock.html** - Standalone analog pace clock with HTML5 Canvas

Features
--------

* Customizable analog clock with 60-second markings
* Optional digital VFD-style time display (HH:MM)
* Configurable colors, sizes, and hand designs
* Optional minute hand for longer intervals
* Screensaver mode with drifting animation
* Full URL parameter configuration system
* Live setup panel for real-time adjustments
* Club logo display
